Separable Equations

Separable Differential Equation \[ {dy \over dx} = f(y)g(x) \rightarrow { {dy \over f(y)}} = g(x)dx \rightarrow {\int {{dy \over f(y)} = \int g(x)dx}}\] Homogenous First Order Simplifications \[ {dy \over dx} = {f({y \over x})} \rightarrow u = {y \over x} \rightarrow {dy \over dx} = u + x{du \over dx} \rightarrow u + x{du \over dx} = f(u) \] \[ {dy \over dx} = {f({ax+by})} \rightarrow u = {ax+by} \rightarrow {du \over dx} = a + b{dy \over dx} \]

Linear Equations

Standard Form \[ {dy \over dx} + p(x)y = q(x) \rightarrow u(x) = e^{\int p(x)dx} \rightarrow y(x) = {1 \over u(x)}({\int{u(x)q(x)ds + c}}) \] Bernoulli's Equation \[ {dy \over dx} + p(x)y = q(x)y^n \rightarrow y^{-2}{dy \over dx} + p(x)y^{1-n} = q(x)\] Linear Coefficent Equations \[ (a_1x+b_1y+c_1)dx + (a_2x+b_2y+c_2)dy = 0 \rightarrow x=u+k, y=v+1\] \[ (a_1u+b_1v+c_1+a_1k+b_1l)du + (a_2u+b_2v+c_2+a_2k+b_2l)dv = 0 \] \[ (a_1u+b_1v)du + (a_2u+b_2v)dv = 0 \]

Exact Equations

\[ M(x,y)dx + N(x,y)dy = 0 \] Exact if: \[ {\partial M \over \partial y} = {\partial N \over \partial x} \]   If exact: \[ F(x,y) = {\int M(x,y)dx} + g(y) \rightarrow {\partial F \over \partial y} = {\partial \over {\partial y} }({\int M(x,y)dx}) + g^{\prime} (y) = N(x,y) \] \[ g^{\prime} (y) = a \rightarrow g(y) = b + c \rightarrow F(x,y) = \int{M(x,y)dx + g(y)} \]

Second Order Linear Equations

Homogenous \[ ay^{\prime \prime} + by^{\prime} + cy = 0 \] \[ar^2+br+c = 0 \]   Solve for \(r_1, r_2\):
    if \(r_1 \ne r_2\): \[ y_h(t) = c_1e^{r_1t} + c_2e^{r_2t} \]     if \(r_1 = r_2\): \[ y_h(t) = c_1e^{r_1t} + c_2te^{r_2t} \]     if \(r_1,r_2 = \alpha \pm i\beta \): \[ y_h(t) = c_1e^{\alpha t}cos(\beta t) + c_2e^{\alpha t}sin(\beta t) \] Non-Homogenous \[ ay^{\prime \prime} + by^{\prime} + cy = f(t) \] \[ y(t) = y_h(t) + y_p(t) \]   Method 1: Undetermined Coefficents
Make a guess based on \(f(t)\) and solve for \(y_p(t)\) coefficents by substituting the guess into the equation. Guesses based on this chart: Link
  Method 2: Variation of Parameters
Use the \(y_1\) and \(y_2\) values found from the homogenous solution to find \(y_p(t)\). \[ y_p(t) = v_1y_1 + v_2y_2\] \[ v_1(t) = -\int{{f(t)y_2(t)}\over{a(y_1(t)y_2^{\prime}(t)-y_1^{\prime}(t)y_2(t))}}dt, v_2(t) = \int{{f(t)y_1(t)}\over{a(y_1(t)y_2^{\prime}(t)-y_1^{\prime}(t)y_2(t))}}dt \]
  Method 3: Reduction of Order Use \(y_1\) or \(y_2\) to find a guess with a function \(u\) and substitute the functions into the equation. This reduces the order to a more solveable form. \[ y_p(t) = uy_1 \] It's gotta be differentiated twice for \(y^{\prime}(t)\) and \(y^{\prime \prime}(t) \).

Cauchy-Euler

\[ ax^2{d^2y \over dx^2} + bx {dy \over dx} + cy = f(x) \rightarrow a{d^2y \over dx^2} + (b-a){dy \over dx} + c = f(e^x) \]   If in the form: \[ ax^2{d^2y \over dx^2} + bx {dy \over dx} + cy = 0 \rightarrow ar(r-1)+br+c=0 \]
  Solve for \(r_1, r_2\):
    if \(r_1 \ne r_2\): \[ y_n(t) = c_1x^{r_1} + c_2x^{r_2} \]     if \(r_1 = r_2\): \[ y_n(t) = c_1x^{r} + c_2x^{r}ln(x) \]     if \(r_1,r_2 = \alpha \pm i\beta \): \[ y_n(t) = x^{\alpha}(cos(\beta ln(x)) + isin(\beta ln(x))) \]

Free Vibrations

\( m \) = mass, \( b \) = damping constant, \( k \) = spring constant \[my^{\prime \prime} +by^{\prime} + ky = 0 \]   if \( b^2 > 4mk \) (overdamped): \[ y(t) = c_1e^{r_1t} + c_2e^{r_2t} \]   if \( b^2 = 4mk\) (critically damped): \[ y(t) = c_1e^{ {-b \over 2m} t} + c_2te^{ {-b \over 2m} t} \]   if \( b^2 < 4mk \) (underdamped): \[ y_h(t) = c_1e^{ {-b \over 2m} t}cos( {\sqrt{4mk-b^2} \over 2m}t) + c_2e^{ {-b \over 2m} t}sin( {\sqrt{4mk-b^2} \over 2m}t) \]     \(A\) = amplitude, \(\phi\) = phase shift \[ c_1 = Asin(\phi), c_2 =Acos(\phi), A = \sqrt{c_1^2 + c_2^2} \]     \( w ={ \sqrt{4mk-b^2} \over 2m } = \) angular frequency, \( T = {{2 \pi} \over w} =\) period, \( {w \over {2 \pi}} = \) frequency \[ y_h(t) = Asin(wt + \phi)e^{ {-b \over 2m} t} \]

Laplace Transformations

\[ L\{f(t)\}(s) = F(s) = \int_0^{ \infty }{e^{-st}f(t)dt} \]   When does a Laplace Transformation exist?
    1. If \( f(t) \) is piecewise continuous on \( [0,\infty] \)
    2. If exists \( M \) and \( t \) such that: \( |f(t)| \le Me^{\alpha t} \) when \( t \ge 0 \)
  Properties of Laplace Transformations
    (i) \( L\{ e^{at}f(t) \}(s) = F(s-a), s > a \)
    (ii) \( L\{ f^{m}(t) \}(s) = s^{m}F(s)-s^{m-1}f(0)-...f^{m-1}(0) \)
        \( L\{ f^{\prime}(t) \}(s) = sF(s)-f(0) \)
        \( L\{ f^{\prime \prime}(t) \}(s) = s^2F(s)-sf(0)-f^{\prime}(0) \)
    (iii) \( L\{ t^nf(t) \}(s) = (-1)^n {d^n \over ds^n}F(s) \)
