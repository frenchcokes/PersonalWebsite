Electric Fundamentals

  \( V = \) Voltage (V), \( I = \) Current (A), \( R = \) Resistance \( (Ω) \), \( P = \) Power (W), \( W = \) Work (J) \[ V = IR \] \[ P = IV \] \[ W = \int_0^{t_f} Pdt \]   \( e_r = \) relative permittivity, \( e = \) permittivity of material, \( e_0 = \) permittivity of vaccum \( = 8.85 * 10^{-12} {F \over m} \) \[ e_r = {e \over e_0} \]   Power depends on direction of current and voltage:
    Current leaves from (+) terminal: \( P > 0 \) power is absorbed
    Current leaves from (-) terminal: \( P < 0 \) power is supplied

Ohm's Law

\[ V = IR\] Positive Sign Convention (PSC) - Current travels from high to low \( (+) \rightarrow (-) \)
  Resistors = Current leaves from (-) terminal
  Sources = Current leaves from (+) terminal
When sign convention not obeyed, \[ P=-IV \]

Kirchoff's Laws

Kirchoff's Current Law (KCL):
  At a node: \[ \sum I_{enter} = \sum I_{exit} \]
Kirchoff's Voltage Law (KVL):
  At a loop (Closed wire path that loops back to beginning): \[ \sum V_{supplied} = \sum V_{absorbed} \] Power Balance:
\[ \sum P_{supplied} = \sum P_{absorbed} \]

Series and Parallel Resistors

Series Resistors
Something is in series when there is a direct path between them with no other branches or elements.
  Resistors in series can be simplified into one: \[ R_{eq} = R_1 + R_2 + ... + R_n \]   The voltage of \( V_n \) can be found with the voltage division rule (VDR): \[ V_n = V * { R_n \over R_{eq}} \]   Voltage sources and resistors in series can be converted to one voltage source: \[ I = { V_{eq} \over {R_1 + R_2 + ... + R_n}} = {V_{eq} \over R_{eq} } \] Parallel Resistors
Resistors are parallel when branches for both resistors lead to the same node on both sides.
  They can be simplified into one resistor with: \[ {1 \over R_{eq}} = {1 \over R_{1} } + {1 \over R_{2}} + .... + {1 \over R_n} \]   The current of \( I_n \) can be found with the current division rule (CDR): \[ I_n = {{I * R_{eq}} \over R_n} \]   Current Sources and resistors in parallel can be converted into one current source: \[ I_{eq} = \sum{ I_{R1} + I_{R2} +...+ I_{Rn} } \]

Delta-Wye Transformations

You can convert circuits to equivalent forms which makes them easier to solve.
  Delta to Wye: \[ R_1 ={ { R_b * R_c} \over {R_a + R_b + R_c} }, R_2 ={ { R_a * R_c} \over {R_a + R_b + R_c} }, R_3 ={ { R_a * R_b} \over {R_a + R_b + R_c} } \]   Wye to Delta: \[ R_a = { {R_1R_2 + R_2R_3 + R_3R_1} \over {R_1} }, R_b = { {R_1R_2 + R_2R_3 + R_3R_1} \over {R_2} }, R_c = { {R_1R_2 + R_2R_3 + R_3R_1} \over {R_3} } \]

Node Voltage Analysis

Need # of nodes - 1 equations to solve.
At each node that's not the ground node you chose:
  1. Write KCL (ie summation of currents at a node)
  2. Write the currents in terms of V and R with the equation:
    Where: \( V_1 = \) voltage current is leaving from, \( V_2 = \) voltage current is entering \[ I = {{V_1 - V_2} \over R} \]     \( V_n \) represents the voltage at the assigned node, and the voltage at the ground node is 0. If there is no current already, choose a direction.
  3. Solve with matrix
Special cases: Current sources don't need to be written in terms of V and R.

Mesh Current Analysis

Mesh: A loop with no smaller loops inside
Can only be applied to planar nodes (cannot be drawn without wires crossing over).
  1. Write and label meshes wherever you can on the circuit
  2. For each mesh, write a KVL. Shared resistors voltage should be written as: \[ V_X = R_x(I_{current mesh} - I_{other mesh}) \]   3. Solve with matrix
Current Sources:
  Edge current sources: They directly give a current value, meaning one less equation to solve.

Supermeshes and Supernodes

A supermesh contains two meshes with a shared current source. It can be obtained by using the same process to get a mesh but around both meshes and ignoring the shared element. A KCL between meshes the supermesh is needed to solve.

Supernodes are used to perform NVA when there are multiple voltage sources. We define it around the voltage source as long as it is not attached to the ground node. To solve, find an equation relating the supernode to the rest of the circuit.
